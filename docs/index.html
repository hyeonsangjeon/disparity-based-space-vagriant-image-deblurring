What this is

The preserved 2013 paper and historical 2009–2011 README figures are research records. The 2026 Python package is a reproducible modern reference reconstruction, not the exact deleted original implementation and not bit-identical to historical outputs.

Living-room restoration walkthrough

Mat3 is an authorized Hyeon Sang Jeon living-room photograph rendered with the selected HPO preset. It has no clean ground truth: the 0.874256 proxy score and 0.919059 proxy SSIM use the registered, denoised noisy auxiliary, not PSNR.

Selected HPO: 12 color labels, disparity threshold 1.0, 17px kernels, 192px patches, Tikhonov 0.001, data weight 60, beta 8, feather sigma 12, unsharp amount 0.3, and guided detail-fusion amount 0.5.

Optional historical article · Block-boundary blending (2011)

Before the journal moved from fixed blocks to disparity-defined regions, the 2011 work documented a PSF-aware way to suppress FFT block ringing and merge restored patches without visible seams.

Boundary conditioning

Blocks were 256 or 512 pixels. The overlap followed the estimated PSF width, while a PSF-half-width dead zone removed the pixels altered only to satisfy FFT periodicity.

2D mask composition

Horizontal and vertical 1D ramps were multiplied into corner masks. Four corner, four edge, and one interior mask then accumulated restored patches so overlapping weights summed to one.

Historical contribution record and implementation evidence

The 2011 IPIU poster Efficient Boundary Artifact Removal in a Block-Based Deblurring Algorithm Using Multiple Images lists Hyeon Sang Jeon first, followed by Chang-Hwan Son and Hyung-Min Park. Its sections 2.2 and Figures 3-5 document linear edge interpolation, a PSF-adaptive dead zone, separable 2D masks, and overlap-based image composition. The preserved 2011 presentation repeats this flow, and the recovered C++ records BoundaryEdgeSmoothing_LinerInterpolation, nine block-position masks, and weighted BlockSumImage accumulation.

This block method is a documented predecessor, not a claim that the 2026 implementation is the deleted original. The 2013 paper's central contribution replaces fixed rectangular blocks with disparity-guided content regions; the modern reference implementation generalizes the merge with normalized feather blending.
